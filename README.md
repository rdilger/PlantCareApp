# Plant Care App

A household plant management application to help track watering, fertilizing, and plant health.

## Project Status
🌱 **Learning Phase** - Exploring low-code/no-code development

## Problem Statement
Managing multiple household plants becomes challenging when tracking different care schedules for watering, fertilizing, and health monitoring. The situation becomes more complex when temporarily delegating plant care to others.

## Target User
Initially: Personal use for household plant management
Future: Multi-user households and care delegation scenarios

## Tech Stack (Planned)
- **Platform**: Glide (mobile-first, low-code)
- **Database**: Glide Tables or Google Sheets integration
- **Notifications**: In-app and email

## Project Goals
1. Learn low-code/no-code development fundamentals
2. Build a functional MVP for personal plant care management
3. Iterate based on real-world usage
4. Keep resource usage minimal during learning phase
